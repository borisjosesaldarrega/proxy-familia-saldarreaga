# ========== NOTAS DE USO ==========
# Esta lista debe agregarse al archivo hosts de tu sistema:
# - Windows: C:\Windows\System32\drivers\etc\hosts
# - Linux/Mac: /etc/hosts
#
# Para aplicar los cambios:
# - Windows: Ejecutar "ipconfig /flushdns"
# - Linux/Mac: Ejecutar "sudo dscacheutil -flushcache" o "sudo systemd-resolve --flush-caches"
#
# Recomendación: Combinar con un bloqueador de contenido como uBlock Origin
# para una protección más completa.
